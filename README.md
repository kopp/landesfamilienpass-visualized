# Landesfamilienpass

(Deutsche Version siehe unten. German version see below.)

This is the code for a website that shows the attractions of the Landesfamilienpass in Baden-Württemberg on an map and a table and allows the user to quickly find interesting attractions and mark favorites.

The data source for the application is
[on the website of the Sozialministerium Baden-Württemberg](https://sozialministerium.baden-wuerttemberg.de/de/soziales/familie/leistungen/landesfamilienpass).

It is converted to json for machine readibility: [`lfp.json`](src/data/lfp.json).

If you know about attractoins that are not mentioned here, please raise an issue or pull request on GitHub.


# Landesfamilienpass (Deutsch)

Dies ist der Quellcode für eine Webseite, die die Attraktionen des Landesfamilienpasses in Baden-Württemberg auf einer Karte und in einer Tabelle darstellt und es ermöglicht, schnell interessante Angebote zu finden und Favoriten zu markieren.

Die Datenquelle der Anwendung ist die [Webseite des Sozialministeriums Baden-Württemberg](https://sozialministerium.baden-wuerttemberg.de/de/soziales/familie/leistungen/landesfamilienpass).

Die Daten wurden zur maschinellen Lesbarkeit in JSON konvertiert: [`lfp.json`](src/data/lfp.json).

Wenn Sie Attraktionen kennen, die hier fehlen, erstellen Sie bitte ein Issue oder einen Pull-Request auf GitHub.
